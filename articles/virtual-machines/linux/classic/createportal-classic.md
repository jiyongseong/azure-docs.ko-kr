# <a name="create-a-custom-virtual-machine-running-linux"></a>Linux를 실행하는 사용자 지정 가상 머신 만들기
> [!IMPORTANT] 
> Azure에는 리소스를 만들고 작업하기 위한 [리소스 관리자 및 클래식](../../../resource-manager-deployment-model.md)이라는 두 가지 배포 모델이 있습니다. 이 문서에서는 클래식 배포 모델 사용에 대해 설명합니다. 새로운 배포는 대부분 리소스 관리자 모델을 사용하는 것이 좋습니다. Resource Manager 모델을 사용하여 Linux VM을 만드는 방법에 대한 자세한 내용은 [포털을 사용하여 Azure에서 Linux VM 만들기](../quick-create-portal.md?toc=%2fazure%2fvirtual-machines%2flinux%2ftoc.json)를 참조하세요.
