# <a name="create-list-or-delete-a-user-assigned-managed-service-identity-msi-using-the-azure-portal"></a>Azure Portal을 사용하여 사용자 할당 MSI(관리 서비스 ID) 생성, 나열 또는 삭제

[!INCLUDE[preview-notice](~/includes/active-directory-msi-preview-notice-ua.md)]

관리 서비스 ID는 Azure Active Directory에서 관리 ID를 Azure 서비스에 제공합니다. 이 ID를 사용하면 코드에 자격 증명을 포함할 필요 없이 Azure AD 인증을 지원하는 서비스에 인증할 수 있습니다. 

현재는 Azure Portal을 사용하여 사용자 할당 관리 ID를 생성, 나열 또는 삭제할 수 없습니다.  업데이트를 다시 확인하십시오.
