#### <a name="to-install-updates-via-the-azure-portal"></a>Azure Portal을 통해 업데이트를 설치하려면

1. StorSimple 장치 관리자를 이동하고 **장치**를 선택합니다. 서비스에 연결된 장치 목록에서 업데이트하려는 장치를 선택하고 클릭합니다.

    ![장치 업데이트](../includes/media/storsimple-virtual-array-install-update-via-portal-04/azupdate1m.png) 

2. **설정** 블레이드에서 **장치 업데이트**를 클릭합니다.

    ![장치 업데이트](../includes/media/storsimple-virtual-array-install-update-via-portal-04/azupdate2m.png)  

3. 소프트웨어 업데이트를 사용할 수 있는 경우 메시지가 나타납니다. 업데이트를 확인하려면 **스캔**을 클릭할 수도 있습니다. 실행 중인 소프트웨어 버전을 기록해 둡니다. 

    ![장치 업데이트](../includes/media/storsimple-virtual-array-install-update-via-portal-1/azupdate3m1.png)

    스캔이 시작되고 성공적으로 완료되면 알림이 표시됩니다.

    ![장치 업데이트](../includes/media/storsimple-virtual-array-install-update-via-portal-1/azupdate5m.png)

4. 업데이트가 스캔되면 **업데이트 다운로드**를 클릭합니다.

    ![장치 업데이트](../includes/media/storsimple-virtual-array-install-update-via-portal-1/azupdate6m.png)

5. **새 업데이트** 블레이드에서 릴리스 정보를 검토합니다. 또한 업데이트가 다운로드된 후에는 설치를 확인해야 합니다. **확인**을 클릭합니다.

    ![장치 업데이트](../includes/media/storsimple-virtual-array-install-update-via-portal-1/azupdate7m.png)

6. 업로드가 시작되고 성공적으로 완료되면 알림이 표시됩니다.

     ![장치 업데이트](../includes/media/storsimple-virtual-array-install-update-via-portal-1/azupdate8m.png)

5. **장치 업데이트** 블레이드에서 **설정**을 클릭합니다.

     ![장치 업데이트](../includes/media/storsimple-virtual-array-install-update-via-portal-1/azupdate11m1.png)

6. **새 업데이트** 블레이드에서 업데이트가 중단되었다는 경고가 나타납니다. 가상 배열이 단일 노드 장치인 경우 장치가 업데이트된 후에 다시 시작됩니다. 따라서 진행 중인 모든 IO가 중단됩니다. **확인**을 클릭하여 업데이트를 설치합니다.

    ![장치 업데이트](../includes/media/storsimple-virtual-array-install-update-via-portal-1/azupdate12m.png)

7. 설치 작업이 시작될 때 알림을 받습니다.

    ![장치 업데이트](../includes/media/storsimple-virtual-array-install-update-via-portal-1/azupdate13m.png)

8.  설치 작업을 성공적으로 완료한 후에 **장치 업데이트** 블레이드에서 **작업 보기** 링크를 클릭하여 설치를 모니터링합니다. 

    ![장치 업데이트](../includes/media/storsimple-virtual-array-install-update-via-portal-1/azupdate15m1.png)

    이 작업을 수행하면 **업데이트 설치** 블레이드로 이동합니다. 이 작업에 대한 자세한 정보가 표시됩니다.

    ![장치 업데이트](../includes/media/storsimple-virtual-array-install-update-via-portal-1/azupdate16m1.png)

9. 소프트웨어 버전 업데이트 0.6(10.0.10293.0)을 실행하는 가상 배열로 시작한 경우 이제 업데이트 1을 실행하여 완료됩니다. 나머지 단계를 건너뛸 수 있습니다. 업데이트 0.6(10.0.10293.0) 이전의 소프트웨어 버전을 실행하는 가상 배열로 시작한 경우 이제 업데이트 0.6으로 업데이트됩니다. 업데이트를 사용할 수 있음을 나타내는 다른 메시지가 표시됩니다. 4-8단계를 반복하여 업데이트 1을 설치합니다.

    ![장치 업데이트](../includes/media/storsimple-virtual-array-install-update-via-portal-1/azupdate17.png)
